BrightFuture Academy - Static School Website Template

Files included:
- index.html
- pages/about.html
- pages/academics.html
- pages/activities.html
- pages/gallery.html
- pages/admissions.html
- pages/contact.html
- css/style.css
- js/main.js
- images/*.svg

How to use:
1. Extract the zip file.
2. Open index.html in a browser.
3. To deploy, upload the full folder to any static hosting service (GitHub Pages, Netlify, Vercel static site, cPanel, Apache, Nginx).
4. Replace placeholder text and SVG images with your school content.

No forms, databases, or backend are required. Bootstrap and icons are loaded via CDN.
